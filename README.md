# Gitdark Theme
### Based on GitHub colorscheme in Dark mode

![Imgur](https://i.imgur.com/n0YXvoa.png)
Installation: Add the folder 'Colors' in your /etc/vim/.
Open your VIM and type: 
<pre>
:colorscheme Gitdark
</pre>
Then press enter.

Languages where the theme is better finished:
<br>
C
<br>
Python
<br>
Ruby
