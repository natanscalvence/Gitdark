# Gitdark Theme
### Based on GitHub colorscheme in Dark mode

![Imgur](https://i.imgur.com/n0YXvoa.png)
### Installation
<pre>
cd /etc/vim/
sudo git clone https://github.com/natanscalvence/Gitdark.git
cd Gitdark
mv colors /etc/vim/
</pre>
### Open your vim than type: 
<pre>
:colorscheme Gitdark
</pre>
Then press enter.
